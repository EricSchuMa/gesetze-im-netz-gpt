Datenschutzerklärung für die Gesetze im Netz API von Max Schumacher

1. Allgemeines

Diese Datenschutzerklärung gilt für die Gesetze im Netz API, betrieben von Max Schumacher. Die API ermöglicht den Zugriff auf verschiedene deutsche Rechtsdokumente. Diese Datenschutzerklärung soll Nutzer und Nutzerinnen über die Art, den Umfang und den Zweck der Verarbeitung von personenbezogenen Daten innerhalb unserer API informieren.

2. Verantwortlicher

Max Schumacher

3. Grundlage der Datenverarbeitung

Die Gesetze im Netz API erhebt oder speichert keine personenbezogenen Daten der Nutzer und Nutzerinnen. Die Nutzung der API ist anonym, und es werden keine Anfragen protokolliert.

4. Ihre Rechte als betroffene Person

Sie haben das Recht, über Ihre personenbezogenen Daten, die Verarbeitung, die Herkunft der Daten sowie den Zweck der Datenverarbeitung Auskunft zu erhalten. Da jedoch keine personenbezogenen Daten gespeichert oder verarbeitet werden, kann dieser Punkt nur der Vollständigkeit halber erwähnt werden.

5. Datenverarbeitung

Die Gesetze im Netz API speichert oder sammelt keine Daten über ihre Nutzer und Nutzerinnen. Es gibt keine Erfassung von IP-Adressen oder anderen personenbezogenen Daten während der Nutzung unserer Dienste. Die Anfragen an die API erfolgen vollständig anonym, und es werden keine Nutzungsdaten gespeichert.

6. Änderungen an dieser Datenschutzerklärung

Wir behalten uns das Recht vor, diese Datenschutzerklärung jederzeit mit Wirkung für die Zukunft zu ändern. Die jeweils aktuelle Version ist direkt über unsere API abrufbar.
